# 🧠 通用认知状态追踪系统（CSTU）

**Cognitive State Tracker Universal v2.0**

一个可以加载到 OpenClaw 的持久记忆型 Skill，帮助你追踪自己的认知状态演化。

## ✨ 功能

- 📊 十维认知状态实时追踪（温度/正价/压力/能量/开放/稳定/掌控感/反思深度/连接感/时间尺度）
- 🔄 自动生成状态快照，历史只增不删
- ⚠️ 主动预警：燃尽、孤立、失控前发出提醒
- 🎯 根据你的状态动态调整 AI 回复风格
- 📈 长期演化弧线追踪

## 🚀 如何使用

1. 复制 [`cognitive-state-tracker-universal.md`](./cognitive-state-tracker-universal.md) 的全部内容
2. 发给任意 OpenClaw，说："加载这个通用认知追踪系统，帮我初始化"
3. 按提示配置你的基线状态（初始值可都填 0.50）
4. 每天发日志/日记，系统会自动每 5 次输入生成一次状态报告

## 📝 命令

| 命令 | 功能 |
|------|------|
| `/state` | 立即分析当前状态 |
| `/snapshot` | 生成状态快照 |
| `/mystatus` | 查看最新状态 |
